# Airbminusb

## User Stories

```
As a user
So that I can use Airbminusb
I would like to be able to sign up

As a user
So that I can list a place
I would like to be able to sign in

As a user
So that I can compare places
I would like to be able to see a list of all places filtered by availability

As a user
So that others can't use my account
I would like to be able to sign out

As a user
So that I can still access my account
I would like to be able to reset my password

As a lister
So that I can attract potential bookers
I would like to be able to list a place

As a lister
So that bookers know what the space is like
I would like to be able to provide details(name, description and price,)

As a lister
So that bookers know when my space is available
I would like to be able to offer a range of dates

As a lister
So that I can accept bookings
I would like to receive notification of any requests

As a booker
So that I can book an available place
I can't send requests for places that have been booked.

As a booker
So that I can stay in a place
I would like to be able to make a request

As a lister
So that I can book out my place
I would like to be accept or reject requests
```
